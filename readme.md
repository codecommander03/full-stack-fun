### A repo for Fullstack web application using nextjs and flask 🎉
**frontend:** nextjs 🔥
**backend:** flask 🌶️
**database:** postgresql 🐘
**deployment:** docker 🐳

## How to run 🚀
1. Clone the repo
2. Run `docker-compose up` in the root directory
3. Open `localhost:3000` in your browser
